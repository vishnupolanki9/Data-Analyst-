#Task 1 - Data Cleaning and Preprocessing

 #Objective
Clean and preprocess the Netflix Movies and TV Shows dataset using Python (Pandas).

 #Tools Used
- Python
- Google Colab

#Steps Performed
- Loaded the dataset.
- Checked dataset structure and data types.
- Identified missing values.
- Removed duplicate records.
- Filled missing values in selected columns.
- Converted the `date_added` column to datetime format.
- Renamed column headers to lowercase with underscores.
- Standardized text formatting.
- Saved the cleaned dataset as `netflix_cleaned.csv`.

#Output
- Cleaned dataset (`netflix_cleaned.csv`)
- Python code (`task1.ipynb` or `task1.py`)
